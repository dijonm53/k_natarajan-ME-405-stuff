Detailed Description

Run experimental and simulated first-order linear response tests and plot the results.

This program utilizes a simple GUI with a plot in it. It uses Tkinter to display the GUI. The experiemental results are taken from the Nucleo, which is received by the serial port. This data is then used to compare it to the theoretical output of the step response.

This file uses a template for the GUI, given by Dr. John Ridgely
